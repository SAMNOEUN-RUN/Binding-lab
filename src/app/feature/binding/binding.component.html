Data Binding in Angular

In Angular, data binding is how you connect the data in your TypeScript component to the HTML template, so the UI can display it or respond to user actions. It makes your app dynamic instead of just static HTML.

Why Data Binding is Important

  • Makes your UI dynamic and interactive.
  • Keeps the component and template in sync.
  • Reduces manual DOM manipulation — Angular handles updates automatically.
Type Syntax Direction Example
Interpolation {{ '{{value}}' }} TS → HTML

Property binding [property]="value" TS → HTML
Event binding (event)="method()" HTML → TS
