Component Basics in Angular

What is a Component?
A component is the building block of Angular apps. Every component has 4 main parts:

  • TypeScript file (.ts) → logic & data
  • HTML file (.html) → template / view
  • CSS/SCSS file (.css/.scss) → styling
  • Metadata → tells Angular how to use the component (@Component decorator)

Why Components are Important

  • Reusability → Build once, use in many places.
  • Separation of concerns → Logic, layout, and styles are grouped together.
  • Maintainability → Easier to manage large apps by splitting into smaller pieces.
